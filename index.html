So what's it all about

Hello I'm Robert Diton and having spent 6 and a half years in the Royal Marines, I felt it was time for a change of pace and a new career. I spent some time umm-ing and arr-ing about where I could see myself in a world outside of the Royal Marines. After much pondering I finally landed on something that has always intrigued me. That something was coding and everything related. And so my coding journey began. I started slow and steady learning HTML and CSS. Just basic mark up and styling until gradually I started to get the hang of things.

From there I decided to structure my learning more by doing a 10 week online intensive course with Codecademy. The 'How to Build Websites From Scratch' course was great for learning large amounts in a relatively short time. By the end of the course I was able to create fully responsive websites utilising Bootstrap and basic JavaScript and JQuery. I also learned the command line and how to deploy local repositories to remote repositories on GitHub. Since then I have not stopped learning and have been practising non-stop to further develop the skills I learned and to learn new skills.
